# 🔐 Password Manager Program

A Python command-line password manager that allows users to store website credentials, view saved passwords, generate simple passwords, and manage password records using a text file.

## 🚀 Features

- Save website passwords
- View all saved passwords
- Delete all saved passwords
- Generate a random password
- Stores passwords in a `.txt` file
- Simple menu-driven interface

## 🛠️ Technologies

- Python 3
- Random Module
- File Handling
- Dictionary
