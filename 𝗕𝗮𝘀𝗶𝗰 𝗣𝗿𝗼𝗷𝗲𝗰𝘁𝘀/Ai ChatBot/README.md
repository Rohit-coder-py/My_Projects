# 🤖 AI ChatBot

A simple command-line AI chatbot built with Google's Gemini API. Users can ask questions and receive short, clear, and informative responses directly in the terminal.

## 🚀 Features

- Chat with Google's Gemini AI
- Clean terminal responses
- Continuous conversation loop
- Prompt optimization for concise answers
- Removes Markdown formatting for better readability

## 🛠️ Technologies

- Python 3
- Google Gemini API
- google-genai
- Environment Variables

## 📂 Project Structure

```
Ai ChatBot/
├── chatbot.py
└── README.md
```

## ▶️ How to Run

```bash
python chatbot.py
```

> Make sure your `GEMINI_API_KEY` environment variable is set before running the project.
