# 💰 Personal Finance Tracker

A Python command-line application that helps users record daily expenses, categorize spending, view summaries, edit or delete expenses, and monitor their monthly budget.

## 🚀 Features

- Add expenses
- Categorize expenses (Food, Travel, Shopping)
- View all expenses in a table
- Spending summary by category
- Edit or delete expenses
- Set and track monthly budget
- Clean menu-driven interface

## 🛠️ Technologies

- Python 3
- PrettyTable
- Lists
- Dictionaries
