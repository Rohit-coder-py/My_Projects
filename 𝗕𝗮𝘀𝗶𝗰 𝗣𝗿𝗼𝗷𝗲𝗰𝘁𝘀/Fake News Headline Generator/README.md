# 📰 Fake News Headline Generator

A Python command-line application that generates random fake news headlines across multiple categories and tones. It combines predefined subjects, actions, and objects to create unique headlines and allows users to save generated headlines to a text file.

## 🚀 Features

- Generate random fake headlines
- Multiple categories (Tech, Politics, Sports, Entertainment)
- Multiple headline tones (Breaking, Funny, Shocking, Dramatic, etc.)
- Mixed random mode
- Typewriter animation effect
- Save generated headlines to a `.txt` file

## 🛠️ Technologies

- Python 3
- Random Module
- Time Module
- File Handling
