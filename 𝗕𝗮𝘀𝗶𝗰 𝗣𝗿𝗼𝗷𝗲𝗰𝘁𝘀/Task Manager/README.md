# ✅ Task Manager

A Python command-line application that helps users manage daily tasks. Users can add, view, update, and delete tasks, with all data stored permanently in a JSON file.

## 🚀 Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Persistent storage using JSON
- Simple menu-driven interface

## 🛠️ Technologies

- Python 3
- JSON
- File Handling
- Dictionaries

```
Task Manager/
├── main.py

```
