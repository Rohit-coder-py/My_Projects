# 🧮 Advanced Calculator

A Python command-line calculator that performs basic arithmetic operations while maintaining a calculation history. Users can view previous calculations and save the history to a text file.

## 🚀 Features

- Addition, Subtraction, Multiplication
- Division and Modulus
- Calculation History
- Save History to `.txt` file
- Handles invalid input
- Prevents division/modulus by zero

## 🛠️ Technologies

- Python 3
- Functions
- Lists
- Loops
- File Handling
- Exception Handling
