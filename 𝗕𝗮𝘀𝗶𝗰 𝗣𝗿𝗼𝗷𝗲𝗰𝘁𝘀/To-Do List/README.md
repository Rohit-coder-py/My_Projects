# ✅ To-Do List

A Python command-line To-Do List application that helps users organize daily tasks by adding, viewing, completing, and deleting tasks through a simple menu-driven interface.

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Completion timestamp
- Simple command-line interface

## 🛠️ Technologies

- Python 3
- Datetime Module
- Dictionaries

```
└── README.md
```

```bash
python main.py
```
