# 🎓 Student Result Manager

A Python command-line application for managing student records, calculating results, assigning grades, and saving student data to a text file.

## 🚀 Features

- Add student records
- View all students
- Generate result cards
- Calculate total marks & percentage
- Automatic grade assignment
- Pass/Fail status
- Search students
- Export records to a text file

## 🛠️ Technologies

- Python 3
- Lists
- Dictionaries
- File Handling
