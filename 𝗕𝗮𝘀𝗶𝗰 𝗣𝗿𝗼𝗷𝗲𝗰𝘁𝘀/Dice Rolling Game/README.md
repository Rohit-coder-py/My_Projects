# 🎲 Dice Rolling Game

A simple Python command-line game that simulates rolling two dice. The program generates random dice values and displays the result based on the user's choice.

## 🚀 Features

- Roll two virtual dice
- Random dice generation
- Interactive command-line interface
- Simple and beginner-friendly

## 🛠️ Technologies

- Python 3
- Random Module

## 📂 Project Structure

```
Dice Rolling Game/
├── main.py
└── README.md
```

## ▶️ How to Run

```bash
python main.py
```

## 📚 Concepts Practiced

- Random number generation
- Loops
- User input
- Conditional statements
