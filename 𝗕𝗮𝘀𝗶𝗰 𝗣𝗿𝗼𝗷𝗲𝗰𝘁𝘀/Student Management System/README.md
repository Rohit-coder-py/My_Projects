# 🎓 Student Management System

A Python command-line application for managing student records. The program allows users to add students, update marks, generate report cards, calculate grades, and store data permanently using a JSON file.

## 🚀 Features

- Add new students
- Update subject marks
- Generate report cards
- Automatic average and grade calculation
- Persistent data storage using JSON
- Prevents duplicate roll numbers

## 🛠️ Technologies

- Python 3
- JSON
- File Handling
- Dictionaries
